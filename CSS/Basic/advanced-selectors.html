Combinatory Selector

1. Descendant Combinatory (Child at any stage)

- Select the Specific Child node from the webpage

Syntex:
.red-p span {
            color: green;
}

2. Child Combinatory (Direct Child Select)

- Direct Child

Syntex:
.red-p > span {
            color: green;
}

3. Adjacent Combinatory (right after sibling)

- element that are on the same level and next after the specific level

Syntex:
.red-p + span {
            color: green;
}

4. General sibling

- all element at the same level

Syntex:
.red-p + span {
            color: green;
}

4. Add selector

Syntex:
.red-p + span {
            color: green;
}
